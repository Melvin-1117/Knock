## Getting Started

This project is a starting point for a Flutter application.

### Android: Google Services

`android/app/google-services.json` is not committed (it contains secrets). To run the Android app:

1. If you don’t have `android/app/google-services.json`, copy `android/app/google-services.json.example` to `android/app/google-services.json` and fill in your Firebase project values from the [Firebase Console](https://console.firebase.google.com/).
2. **Use your API key:** Open `android/app/google-services.json` and replace `YOUR_GOOGLE_API_KEY` with your Google Cloud API key (the one you created and restricted to this Android app). Save the file.

Never commit `google-services.json` or real API keys.
